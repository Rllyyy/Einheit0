Überschriften werden mit dem Tag h+Zahl (1-6) hinzugefügt

Eine Liste mit Bulletpoints:
  • Wird mit den Tag ul (unordered list) initiiert
  • Einzelne Items einer Liste werden mit dem Tag li hinzugefügt
Eine Liste mit Zahlen:
  1. Eine Liste mit Zahlen werden mit dem Tag ol (ordered list)
  2. Einzelne Items einer Liste werden mit dem Tag li hinzugefügt
Bilder hinzufügen:
  • Bilder werden mit dem Tag git img src = "String" hinzufüget
  • Im Image Tag können weitere Attribute angegeben werden (height/width)
HTML Befehl Auswirkung
table Initiiert eine Tabelle
tr Erzeugt eine neue Spalte (Table row)
th Setzt die Spaltenüberschriften fest (Table Header)
td Befüllt die Spalte (Table Data)
Forms:
  • Forms werden mit dem Tag form initiiert
  • input: type="text" placeholder = "Beispiel" name="name"
  • button: wird mit dem Button Tag aufgerufen

Im Header können Definitionen für z.B. alle h1 angelegt werden oder man verlinkt diese im Header

CSS Befehle:
  • color
  • background-color
  • width
  • height
  • margin für Außerhalb des Elements
  • padding für Formatierung innerhalb des Elements
  • Fonts
  • Border mit px und Umrandung
